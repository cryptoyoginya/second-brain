# Second Brain

Личная база знаний — вики моих интересов. Построена по паттерну Андрея Карпатого
(LLM Knowledge Base): LLM инкрементально строит и поддерживает персистентную markdown-вики,
а не переоткрывает знания с нуля при каждом запросе.

Приватный репозиторий. Открывается как **Obsidian vault**. Не связан с другими проектами.

## Слои

- `raw/` — сырые источники (статьи, заметки, Telegram, YouTube). **Неизменяемо.**
- `wiki/` — вики, которую строит и поддерживает LLM. Точка входа — `wiki/index.md`.
- `outputs/` — отчёты и ответы.
- `CLAUDE.md` — схема системы для агента.

## Поток работы

1. Кладёшь источник в `raw/` (Web Clipper, копипаст, экспорт Telegram, yt-dlp).
2. Говоришь агенту: «Обработай новое в `raw/`» → операция **Ingest**.
3. Задаёшь вопросы по накопленному → **Query**.
4. Раз в месяц — **Lint** (противоречия, сироты, пробелы).

Подробности и конвенции — в [CLAUDE.md](./CLAUDE.md).
