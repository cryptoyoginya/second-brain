# Креативные техники генерации идей

Каталог рабочих методов, когда классический брейншторм застрял. Не для заучивания — как полка инструментов: под задачу берёшь один. Развёрнуто разбирается только [[CRAFT — генерация продуктовой идеи|CRAFT]].

## Суть
- **Double Diamond** (British Design Council, 2005) — чередование дивергентного и конвергентного мышления. Discover → Define (*designing the right thing*), Develop → Deliver (*designing things right*). Есть Triple Diamond (+ ромб масштабирования).
- **ТРИЗ** (Альтшуллер, 1940–60е) — любая задача сводится к **противоречию**, которое решается принципами из анализа тысяч изобретений.
- **SCAMPER** — акроним 7 приёмов: Substitute, Combine, Adapt, Modify, Put to another use, Eliminate, Reverse. Систематизирует генерацию.
- **CRAFT** (Василий Лебедев, ИКРА, 2010е) — работа через «фреймы» и смену способа взаимодействия. Пример: баннер → баннер-лотерейный билет. → [[CRAFT — генерация продуктовой идеи|отдельная страница]].
- **Латеральное мышление** (Эдвард де Боно, 1967) — «боковой заход», увидеть задачу под новым углом. Шесть шляп, провокация, инверсия, аналогия, случайный символ.
- **Theory of Constraints** (Голдратт, 1980е) — найти корневую проблему (древовидная структура), прорывное решение, разрешить конфликты, пошаговый план. Книги «Цель» и «Цель 2.0».
- **Обратный брейншторм** — сначала генерируем идеи, которые **усугубляют** проблему, потом переворачиваем их в решения. Вытаскивает из колеи, где обычный штурм буксует.

## Подробнее
Техники делятся по логике. **Дивергентные** (SCAMPER, латеральное, обратный штурм) — расширяют поле, ломают привычную рамку. **Структурные** (ТРИЗ, ТОС) — сводят задачу к противоречию/корню и бьют по нему. **Процессные** (Double Diamond, CRAFT) — организуют весь маршрут от проблемы к проверенному решению. Под большинство продуктовых задач достаточно одной процессной + одной дивергентной.

> [!tip] Что запомнить
> Три семейства: дивергентные (расширить поле — SCAMPER, обратный штурм), структурные (свести к противоречию/корню — ТРИЗ, ТОС), процессные (маршрут проблема→решение — Double Diamond, CRAFT). Бери по одной из нужных.

> [!check] Практический вывод
> Застрял на идее — не «подумай ещё», а смени инструмент под тип затыка: не видишь новых ходов → обратный штурм / случайный символ; тонешь в симптомах → ТОС (найди корень); нужен полный маршрут до прототипа → CRAFT.
