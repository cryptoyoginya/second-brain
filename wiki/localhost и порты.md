# localhost и порты

**localhost** — твой компьютер, обращающийся сам к себе. «Личный мини-интернет», видимый только тебе: там крутится проект, пока ты его пробуешь, до того как показать миру.

> [!quote]
> localhost — кухня, где пробуешь блюдо до подачи в зал. Порт — номер двери на этой кухне.

## Суть
- **localhost** — локальный запуск: сервер работает на твоей машине, снаружи не виден. Место, где тестируешь до деплоя.
- **Порт** — номер «двери» (`localhost:3000`). Один сервер = одна дверь. Ошибка `EADDRINUSE` = дверь занята (сервер уже запущен на этом порту).
- **3000** — конвенция Next.js/Node. Разные проекты = разные порты; разные страницы одного проекта = разные *пути* (`/landings/crm`), а не порты.

## Подробнее
- Путь запуска проекта, где всё это встречается: проверить окружение (`git --version`, `node --version`, `pnpm --version`) → `git clone` → `pnpm install` (поднимаются и [[Фазы пайплайна и гейты|git-хуки/гейты]]) → `pnpm dev` → превью на `localhost:3000/...`.
- Частая путаница: порт vs путь. Порт — какой *сервер*; путь — какая *страница* внутри него. `EADDRINUSE` лечится не «перезапуском», а пониманием, что сервер уже поднят.

> [!tip] Что запомнить
> localhost — твой компьютер сам себе, виден только тебе. Порт — дверь (`:3000`), один сервер = одна дверь, занята → EADDRINUSE. Разные страницы = разные пути, не порты.

> [!check] Практический вывод
> Увидел `localhost:3000` — это локальное превью, не «выложено в интернет». `EADDRINUSE` — сервер уже запущен, не поднимай второй. Разные проекты запускай на разных портах, страницы одного проекта различай путём.
