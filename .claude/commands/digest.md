Собери дайджест по базе (CLAUDE.md → операция Digest). Только по вики, без внешних данных о состоянии. Три блока, коротко:

1. **Последнее изученное** — 5 последних тем-страниц по `wiki/log.md` и датам, новее сверху. Гранулярность — конкретные страницы-концепты (напр. «Working Backwards», «Воспринимаемая ценность»), НЕ крупные блоки-кластеры. Формат строки: `N. **Имя темы** — [открыть](obsidian://open?vault=second-brain&file=<url-encoded имя без .md>)` (кириллицу кодировать через python urllib.parse.quote).
2. **Давно не повторяла** — 2-3 страницы/куста, к которым давно не возвращалась (старый `updated`, нет свежих входных ссылок) и которые стоит освежить — с одной строкой «почему сейчас».
3. **Бэклог** — топ-2-3 неизученных (`- [ ]`) из `wiki/Бэклог.md`, по приоритету (🔥 → 📌). Если бэклог пуст — пропусти блок молча.
4. **Что изучить дальше** — 2-3 Scout-рекомендации под Фокус/интересы из `wiki/Обо мне.md`: реальные источники (граундить веб-поиском, не выдумывать) или нераскрытые темы из блоков «Дополнительно изучить». Не дублируй то, что уже в бэклоге.

Без воды, без советов по жизни. В конце спроси, что развернуть.
